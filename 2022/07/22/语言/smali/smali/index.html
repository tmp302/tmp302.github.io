java转smali

idea 安装 java2smali 插件

原理

java转smali：

1、使用 javac 将 .java 文件转换成 .class 文件

2、使用 dx.jar 将 .class 文件转换成 .dex 文件

3、使用 baksmali 将 .dex 文件转换成 .smali 文件

smali转java：

1、使用 smali.iar 将 .smali 文件转换成 .dex 文件

2、使用 dex2iar 将 .dex 转换成 .class 文件

3、使用 jad 将 .class 转换成 .java 文件

定义数据类型

参数 描述
const-string 字符串
const/4 最大只允许存放4位数值（4个二进制位）（int、char、byte、short、long、boolean）
const/16 最大只允许存放16位数值（int、char、byte、short、long、boolean）
const/32 最大只允许存放32位数值（int、char、byte、short、long、boolean）
const/high16 最大只允许存放16位数值（float）
const-wide/high16 最大只允许存放16位数值（double、long）
const-class 字节码对象赋值

C char 字符型
I int 整形
J long 长整型 64 位
F float 浮点型
D double 双精度型 64 位

L+全类名路径用/分割 objec

寄存器

android 变量都是存放在寄存器中的，一个寄存器可存储 32位 长度的类型，其中 long 和 double 是 64位 的，需要使用 两个寄存器 保存。

命名规则

参数 描述
v0-n 本地寄存器
p0-n 参数寄存器

v命名法

如有两个局部变量则用 v0、v1 表示，此时如果还有 this 和其他形参，则用 v2-n 表示

p命名法

参数 描述
p0 this
p1 第一个参数
p2 第二个参数
