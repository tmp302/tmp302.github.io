发表于2022-05-20|更新于2022-05-23|Golang

常用

WaitGroup 控制

参数 描述
Add() 等待组的计数器 +1
Done() 等待组的计数器 -1
Wait() 当等待组计数器不等于 0 时阻塞直到变 0。

channel

Mutex 互斥锁

加锁后需解锁后才能继续加锁

参数 描述
Lock() 加锁
Unlock() 解锁

RWMutex 读写锁

可加多个读锁或一个写锁

写锁权限高于读锁，有写锁时优先进行写锁定

参数 描述
RLock 读锁
Lock() 写锁
Unlock() 解锁
