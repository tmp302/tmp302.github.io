命令执行

exec()

用于执行外部程序，并返回最后一行输出。

shell_exec()

用于执行 shell 命令，并返回输出。

system()

用于执行外部程序，并将输出直接打印到标准输出。

passthru()

用于执行外部程序，并将输出直接传递给标准输出。

popen()

用于执行外部程序，并创建一个管道连接到该程序的输入或输出。

代码执行

eval()

用于解析并执行字符串中的 PHP 代码。

assert()

用于创建匿名函数并执行其中的代码。

create_function()

用于创建匿名函数并执行其中的代码。

include() 或 require()

用于包含并执行指定文件中的 PHP 代码。

include_once() 或 require_once()

用于包含并执行指定文件中的 PHP 代码，但只包含一次，防止重复包含。

文件上传

move_uploaded_file()

将上传的文件移动到指定位置。

upload_tmp_dir()

用于设置上传文件的临时目录。

is_uploaded_file()

检查文件是否是通过 HTTP POST 上传的。

文件读取

file_get_contents()

将整个文件读取为字符。

fread()

从打开的文件中读取指定长度的数据。

fgets()

从打开的文件中逐行读取数据。

file()

将文件的每一行读取到数组中。

文件写入

file_put_contents()

将字符串内容写入文件。

fwrite()

向打开的文件中写入数据。

file()

将数组的每个元素写入文件的每一行。

fprintf()

将数组的每个元素写入文件的每一行。

序列化

serialize()

将 PHP 值序列化为字符串。

unserialize()

将序列化的字符串转换回 PHP 值。

json_encode()

将 PHP 值转换为 JSON 字符串。

json_decode()

将 JSON 字符串转换为 PHP 值。

忽略报错

error_reporting()

设置报错级别。通过设置报错级别为 0，可以忽略所有报错信息。

ini_set()

设置 PHP 配置项。通过设置 display_errors 为 off，可以禁用报错信息的显示。

@

将其放置在表达式前面，可以忽略该表达式的报错信息。
