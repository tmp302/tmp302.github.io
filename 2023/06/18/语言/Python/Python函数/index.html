命令执行

os.system(command)

在操作系统中执行命令，并返回执行结果的状态码。

os.popen(command)

执行命令并返回一个文件对象，可以用于读取命令的输出。

subprocess.call(args)

在新的子进程中执行命令，并等待命令完成。

subprocess.check_output(args)

在新的子进程中执行命令，并返回命令的输出结果。

代码执行

eval()

执行字符串中的 Python 表达式，并返回结果。

exec()

执行字符串中的 Python 代码。

compile()

将字符串编译为可执行的 Python 代码对象。

文件读取

open()

用于打开文件并返回文件对象。

read()

用于从文件对象中读取数据。

readline()

用于从文件对象中逐行读取数据。

readlines()

用于从文件对象中读取所有行并返回一个列表。

文件写入

write()

将指定内容写入文件。

writelines()

将字符串列表逐行写入文件。

print()

配合文件对象：将内容写入文件。

序列化

pickle.dumps()

将 Python 对象序列化为字节流。

pickle.loads()

将序列化的字节流反序列化为 Python 对象。

json.dumps()

将 Python 对象转换为 JSON 字符串。

json.loads()

将 JSON 字符串转换为 Python 对象。
